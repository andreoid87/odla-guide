# ODLA GUI Guide

Interactive visual guide for the ODLA AppConfigurator and related dialogs.

## Contents

This repository contains a static site built with plain HTML, CSS and JavaScript.
The guide is designed for GitHub Pages and does not require a build step.

## Local Preview

Open `index.html` directly in a browser or serve the repository root with any static file server.

## Publishing

GitHub Pages can publish this repository directly from the `main` branch root.

## Scope

Current guide includes:
- main AppConfigurator overview
- physical keys area
- new/edit app dialogs
- new/edit binding dialogs
- swap binding dialog
- new/edit command button dialogs
- import/export configuration dialogs

Current guide intentionally excludes:
- HID calibration window
- delete confirmation message boxes
- inline menu rename flow (`Nuovo menu`)
- `Collega comando` dialog, because it is present in code but not currently wired from the UI
